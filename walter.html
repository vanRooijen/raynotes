Voor ek jou verloor will ek net gou twee baie algemene concepte verduidelik.

Wat is 'n programming language?

Dit is eenvoudig net 'n taal waarmee jy met die rekenaar praat. Julle gaan probably eerste van Object Pascal leer. Maar daar is vreeslik baie tiepes programming languages.

Wat is 'n IDE (Integrated Development Environment)?

Dit is die program waarin jy met programming languages, 'n program skryf. Jy kry verskillende IDE's vir verskillende programming languages, maar jy kry ook IDE's wat kan programme maak in vreeslik baie verskillende programming languages. Julle gaan as julle die programming language Object Pascal leer, eerste Lazerus gebruik as 'n IDE. Later gaan julle Delphi gebuik.
